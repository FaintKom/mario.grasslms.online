    Source ingestion

    What happens

    Source files (PDFs, markdown, plain text, scraped JSON) get queued for processing. Each file gets a stable hash so re-runs skip work that's already done.

    Why it matters

    Distillation costs LLM tokens — re-processing the same file is throwing money away. The hash check makes the pipeline idempotent: rerun a thousand times, the bill stays at the first run.
