# Design log — Sales Kickoff Facilitator Guide (90 min)

> 90-minute facilitator-led workshop, paired with the FinTech B2B Card Onboarding SCORM module. Companion artefact extending self-paced microlearning into team-context practice. JD requirements closed: facilitator guides, participant workbooks, adult-learning principles in remote-first orgs, ADDIE / SAM application, blended learning solutions, sales-enablement context.

---

## Open questions at start

- **Q1.** Live in-room vs virtual vs hybrid? Each changes role-play logistics.
- **Q2.** How tight a script — verbatim talk track for the facilitator, or principles + flexibility?
- **Q3.** Does the cohort take this *after* the SCORM module (assume base) or *as the only thing* (must front-load product context)?
- **Q4.** Length — 90 min is a normal kickoff slot, but pushes through the 20-min adult-attention ceiling. Break logic?
- **Q5.** Print PDF, web HTML, or both? Field-facilitators print; remote facilitators screen-share.

## References pulled

- **Bob Pike — Creative Training Techniques (5th ed., 2003+)** — 90/20/10 rule (no module >90 min, change activity every 20 min, revisit material every 10 min). Direct constraint on block structure.
- **Bersin / Josh Bersin Academy — *Facilitator's Field Guide* patterns** — modern enablement guide conventions: agenda timeline, materials box, debrief Qs, common pitfalls.
- **Sharon Bowman — *Training from the Back of the Room* (2008)** — 6 trumps: movement > sitting, talking > listening, images > words, writing > reading, shorter > longer, different > same. Set the active-block bias.
- **Donald Schön — *The Reflective Practitioner* (1983)** — reflection-in-action; informs post-role-play debrief prompts.
- **Patti Shank — *Practice and Feedback for Deeper Learning* (2017)** — feedback timing + specificity rules.
- **David Kolb — Experiential Learning (1984)** — CE → RO → AC → AE; the kickoff is *the* place to close the loop the SCORM started.
- **Cathy Moore — Action Mapping** — every block tied to a concrete behaviour the rep will do on Monday.
- **70-20-10 (Charles Jennings)** — this workshop is the "20 %" (social / peer) layer. SCORM = 10 %, on-job calls = 70 %.
- **Kirkpatrick L1–L4** — workshop is the L2 + L3 transition zone.

## 1 · Analyse

**Audience persona.** Mixed-tenure SDR / AE cohort, 8–14 reps per session, ~25–40 y. Some did the SCORM module 24–48 h ago (high recall); some skipped or did it weeks back (rusty). Mixed video-fatigue level if virtual. Some have already had 3–5 live SMB calls; some haven't called yet. Energy: variable — facilitator's first job is to calibrate, second is to teach.

> **Note — persona is illustrative**, drawn from general B2B sales-onboarding patterns. Not based on access to any specific company's cohort data. In production this would be refined in week 1 from real cohort tenure mix + ramp-velocity data.

**Performance context.** This workshop sits between the self-paced SCORM module (L2 learning evidence) and the rep's first 30 live calls (L3 behaviour evidence). It is the *bridge* — the place where the rep practises the moves with their peers and a coach watching, before facing a real SMB owner.

**Constraints.**
- 90 min total (sales kickoff slot is fixed company-wide).
- 8–14 reps per session (cohort logistics).
- Facilitator = enablement lead OR senior AE OR external coach — script must read clean to all three.
- Must work in-room *and* virtual (Zoom / Meet). Hybrid is harder; deferred to v2.
- Must print clean as PDF for in-room facilitators who walk the room with paper.

**Prior-art scan.** Most sales kickoff agendas in the wild are deck + Q&A — facilitator narrates, learners watch. That violates Bowman's 6 trumps and Pike's 90/20/10. Best-in-class (HubSpot Solutions Partner Academy, MEDDIC Academy live workshops) flip the ratio: 20 % facilitator talk, 60 % rep practice, 20 % peer + coach feedback. Adopting that ratio here.

## 2 · Design

### 2a · ADDIE.Design — chose SAM-leaning hybrid

- **ADDIE** rejected as primary frame — sales kickoff content rots in weeks (product / pricing / segment shifts), so a phase-gated waterfall is wrong shape.
- **SAM** picked — design assumes the guide will iterate after each cohort runs it. The agenda blocks are sized so any single block can be swapped without rewriting the others.
- **4C/ID** applied at block level (each role-play block = whole-task) but not at workshop level (workshop is too short to do full 4C/ID cycle).

### 2b · 4C/ID architecture (block-level)

| Component | In this workshop |
|---|---|
| **Learning Tasks** | Block 2 — re-rehearse positioning openers as a group. Block 3 — role-play trio (rep / buyer / observer) on assigned objection scenarios. |
| **Supportive Information** | Workshop opening recap of María persona + 4 props. Sidebar callouts in the guide naming the framework being applied. |
| **Procedural Information** | Block 2 talk-track cards (printed handout — each rep gets one). Mirror of SCORM Screen 4 content, but as a *physical artefact* the rep practises with. |
| **Part-task Practice** | Block 2 partner drill — A says trigger, B fires correct opener, switch. Drill to automaticity. |

### 2c · Kolb experiential cycle mapping

| Kolb stage | In this workshop |
|---|---|
| **Concrete Experience** | Block 3 role-play trios — reps DO the conversation, observer notes. |
| **Reflective Observation** | Block 3 debrief — observer shares pattern noticed, rep self-reflects on one move that worked + one that didn't. |
| **Abstract Conceptualisation** | Block 2 talk-track + Block 4 facilitator names the rule ("re-state objection back first, *then* deliver substance"). |
| **Active Experimentation** | Block 4 commitment cards — "next 3 calls I will try X" — and re-test in next workshop / 1:1. |

### 2d · Kirkpatrick reverse from L4

| Level | Metric | Instrument in workshop |
|---|---|---|
| **L4 — Results** | Same as SCORM module: SMB ramp 90 d → ≤67 d; win-rate +5 pts within 6 months | CRM, segment-tagged outcomes |
| **L3 — Behaviour** | Talk-track adherence first 5 calls ≥80 %; objection-handling rubric ≥3.5/5 at week 4 | Manager observation rubric (handed out at workshop end). Call-recording sample at week 4. |
| **L2 — Learning** | Role-play observer rubric ≥3.5/5 by end of workshop on objection handling + opener fluency | Observer rubric form printed in this guide (workbook page). |
| **L1 — Reaction** | "I am more confident on calls this week than last week" ≥4.2/5 | Anonymous Mentimeter / sticky-note pulse at end of workshop. |

### 2e · Modern LXD methods applied

- **Bob Pike 90/20/10** — workshop = 90 min; activity change every 20 min minimum; key concepts revisited 3× across blocks 2 + 3 + 4.
- **Sharon Bowman 6 trumps** — every block has movement (in-room: trio shuffle; virtual: breakout rooms), talking, writing (commitment cards), images-on-handout, short slot, different framing each round.
- **Action Mapping (Cathy Moore)** — every block named by *action the rep performs*, not topic. "Drill openers" not "Positioning theory".
- **70-20-10** — this workshop is the 20 % (social / peer-learned) layer. SCORM = 10 %; on-job = 70 %.
- **Performance consulting (Hansgaard / Robinson)** — commitment cards in Block 4 force the L3 bridge from workshop to job. Without them, L2 evidence stays in the workshop.
- **Schön reflection-in-action** — Block 3 trio includes an observer role explicitly so reflection happens *during* the practice, not only after.
- **Patti Shank feedback rules** — observer feedback uses 1-positive / 1-specific-improvement format, not vague "good job".

## 3 · Develop — build log

- **Decision: HTML print-ready over Word / Google Doc.** Web-printable HTML stays in DS-token consistency with the pipeline page; facilitator can also screen-share the URL in virtual sessions.
- **Decision: 5 blocks not 4.** First sketch had 4 blocks of ~20 min each. Pike's "change every 20" said push to 5 blocks of 12–30 min varying length, so attention re-engages each switch. Final shape: 5 + 15 + 25 + 30 + 12 + 3 = 90.
- **Decision: trio role-play not pair.** Pair role-play loses the observer — no third eye, no rubric, no specific feedback. Trio: rep / SMB owner / observer with rubric. After each round, rotate.
- **Decision: include observer rubric in the guide itself.** Printable / detachable workbook page. Closes the loop from workshop L2 evidence into something measurable.
- **Decision: virtual-adaptation sidebars in each block.** Same block, two paths: in-room steps vs virtual steps. Avoids two separate guides for the same workshop. ~3 sidebars total.
- **Decision: commitment cards as the L3 bridge.** Block 4 has each rep write 3 specific moves they'll try on their next 3 calls. Manager 1:1 reviews them at week's end. This is *the* mechanism that turns the workshop from theatre into behaviour.
- **Decision: keep facilitator script semi-loose.** Verbatim openers + cue questions, but no word-for-word script for the body. Senior facilitators reject scripts; junior facilitators need cues. Balance via blocked "say something like…" framing.
- **Failure expected: virtual breakouts.** In Zoom, 8–14 reps split into trios = 3–5 breakouts. Coach can't be in all rooms. Mitigation: each trio elects an observer who fills the rubric, photo / paste it into shared sheet. Logged as a real-world friction.
- **Decision: pre-work checklist.** 24 h before — facilitator confirms SCORM completion on cohort, prints handouts, room or Zoom link. 30 min before — opens cards. Reduces day-of friction.

## 5 · Evaluate — metric plan

| Metric class | Target | How measured |
|---|---|---|
| **L1 — Reaction** | "Confidence higher than last week" ≥4.2/5; "Time well-spent" ≥4.5/5 (workshop > module bar because synchronous time is more expensive) | Anonymous in-room (sticky) or Mentimeter pulse at end |
| **L2 — Learning** | Observer rubric mean ≥3.5/5 on opener fluency + objection handling by end of session | Observer rubric (printed in guide, photo-captured to shared sheet) |
| **L3 — Behaviour** | Talk-track adherence ≥80 % on first 5 post-workshop calls; commitment-card follow-through ≥60 % of stated moves attempted within 7 d | Manager 1:1 review of commitment card at +7 d; call recording rubric at +14 d |
| **L4 — Results** | Joint with SCORM module L4 (ramp + win-rate) — workshop is part of the same intervention | CRM, ramp-time field, win-rate by segment |
| **Engagement** | All trio rounds completed (no skipped rotation); no rep silent for >3 min in any block; commitment card 100 % filled | Facilitator log + observer rubrics + commitment card collection |
| **Authoring quality** | Guide refresh ≤90 days; ≤2 facilitator-reported errors per cohort run | Issue log + facilitator post-cohort form |

## 6 · Reflect

**What worked (in design — not yet field-tested)**
- 5-block agenda survives Pike 90/20/10 cleanly.
- Trio structure forces an observer role + rubric — converts subjective "how did it feel" into measurable L2 evidence inside the workshop.
- Commitment cards are the cheapest possible L3 bridge — costs the workshop 12 min, returns weeks of behavioural follow-through.
- Virtual sidebar pattern lets the same guide serve both delivery modes without duplication.

**What broke / surprised in design**
- Original sketch had a "manager debrief" block at the end (Block 5). Pulled it — manager isn't usually in the workshop. Moved manager handoff to a post-workshop checklist instead.
- Initial draft had facilitator script verbatim throughout — felt over-engineered and patronising to senior reps. Rewrote as cue-and-substance: "Say something like X. Then ask Y." That preserves the substance without sounding scripted.

**What I would change next iteration**
- Add a 60-second video clip at workshop start, of an SMB owner saying the most common objection out loud. Models the *real* tone reps will face. Currently rely on facilitator role-play to set this; a clip would be richer.
- Add cohort pre-survey 48 h before — what's your single biggest call anxiety? — and tailor Block 3 scenarios live. Currently blocks are static.
- Build a Slack-template message for the manager so the commitment card follow-up is one click, not a custom write each time.

**Open questions for stakeholder**
- Is the cohort already split by tenure (newhire vs ramped) or mixed? Mixed = adjust pairing rules in Block 3.
- Is there a Gong / Modjo / Chorus call-recording stack? Changes the L3 measurement instrument.
- Who owns the L4 ramp-time metric — RevOps, Sales Ops, Enablement? Need owner to commit to baseline.
