L3 Behaviour · on-the-job moves the intervention should produce

Critical behaviours (from Action Mapping).

  1. Rep uses acknowledge → opener → question on every objection.
  2. Rep matches trigger phrase to correct value prop on ≥80 % of calls.
  3. Rep ends every discovery call with a named next step + written agenda.

Required drivers (New World Kirkpatrick). Mechanisms that move L2 to L3 — without them, learning stays in the workshop.

  • Workshop Block 5 commitment cards (photo → manager within 24 h).
  • Manager 1:1 review of commitment card at +7 d.
  • Job aid (Top-7 SMB objections cheat sheet) on rep's desk.
  • Call-recording rubric review at +14 d using the workshop observer rubric.

Risks + mitigations

L4 attribution. SMB outcomes can move from pricing / product / market causes. Mitigation: track a control cohort that did not take the intervention; report deltas, not absolute lifts.

Manager bandwidth for L3. 1:1 commitment-card review is the bottleneck. Mitigation: 15-min template provided in facilitator guide; Slack auto-prompt at Day 6.

No ramp baseline. Then instrument baseline first; do not run this intervention with no reference number.

No call-recording stack. Mitigation: manager-observation manual rubric (same rubric as workshop observer) — lower N but same instrument.

Cost / value · order of magnitude

Build cost (one-time) — ~10–15 h ID time (initial design) · ~3 h facilitator prep · LMS / authoring-tool licence amortised per module (e.g. Articulate $1.4K / yr ÷ 10 modules = $140 per module).

Run cost per cohort — 90 min × N reps workshop time (opportunity cost ≈ 1.5 h lost calling × $150 / h fully-loaded ≈ $225 / rep) · 1 h facilitator time · ~30 min manager 1:1 × N reps · spaced retrieval LMS hosting amortised.

Value if targets hit — 23 days of ramp recovered × N reps × commission / quota velocity per ramped rep. Order of magnitude $5K–$15K per rep at typical SMB SaaS economics. Net of run cost above: $4K–$13K per rep.
