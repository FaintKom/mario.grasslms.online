Why this artefact: Phase 4 hands the program from the design team to the LMS admin and the manager population without designer dependency (brief §3, master-plan §4). ADDIE Implement gate criteria require an executable runbook + dated launch comms + tested manager calibration plan.

ADDIE Phase 4 · Implement

1 · Phase intent

Cohort 1 goes live 1 July 2026. Slipping past 15 July triggers the sales-org kickoff blackout and pushes launch to September (brief §3, §8). Phase 4 closes that risk by producing four operational artefacts a non-designer can execute:

Deliverable Owner at execution Brief tie-in
launch-comms-plan.md K.M. (sponsor) sends sponsor email; A.S. forwards manager pre-read; designer pre-stages welcome DM §8 sponsor framing · §18 brand voice
lms-deployment-runbook.md LMS admin (Sana) · designer is backup for first 2 cohorts §7 Sana/SCORM 1.2 · §16 tooling
cohort-1-launch-checklist.md Designer drives Day -14 → Day 0; A.S. + 9 pod managers carry Day +14 → Day +120 §5 measurement cadence · §3 hard date
manager-calibration-session-plan.md A.S. facilitates with K.M. observing; designer holds the rubric §8 manager layer · §10.8 manager ask

2 · Phase 4 gate criteria (from master-plan §4)

  • [x] LMS admin can execute the deployment runbook without designer assistance (smoke-test + rollback documented).
  • [x] Manager calibration plan tested with A.S. proxy walk-through; week-2 outcomes measurable (inter-rater variance ≤1 point).
  • [x] Launch comms dated against the 1 July cohort-1 live date; each comm copy-paste-ready in brand voice.
  • [x] Named owner + backup + escalation path on every checkpoint.

3 · Non-goals for Phase 4

  • Building L1/L2/L3/L4 instruments — those live in Phase 5 (05-evaluate/). Phase 4 references them but does not produce them.
  • Producing the rubric content — produced in Phase 2 (02-design/rubric-design.md). Phase 4 ships the calibration session that lands it in managers' hands.
  • Re-training the 9 pod managers (K.M. §8 verbatim: "don't try to retrain managers; give them a shared rubric and two calibration sessions"). D-007 in decisions-log.md is the anti-leaderboard / anti-surveillance corollary.

4 · Hand-off triggers

50 % payment tranche releases at Phase 4 gate sign-off (brief §3). D.R. (Head of People) co-signs against the four deliverables above + critic-agent PASS in 00-project-management/critic-log.md.
