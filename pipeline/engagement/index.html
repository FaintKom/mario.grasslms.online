The engagement walks the seven phases of the pipeline applied to a single SDR onboarding brief. Every deliverable references the same engagement brief, the same named stakeholders (anonymised initials), the same numbers, the same scope. Each phase ends with a sign-off summary that the sponsor signs before the next phase opens.
