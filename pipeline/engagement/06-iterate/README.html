Why this artefact: Brief §21 defines the engagement close-out state (4 cohorts, ramp ≤71 d, retention ≥92 %, rubric live in 9 pods). Brief §20 row "Sponsor turnover" rates K.M.'s departure risk as Low/High — every decision must survive a successor. This phase stands up the post-launch iteration loop and the in-house hand-off so the program keeps improving between cohort waves, with or without K.M. in the chair. Frameworks: ADDIE Phase 6 (Branson 1975 / Molenda 2003) + Phillips Phase-7 iteration logic + Kirkpatrick L1–L4 closed-loop evaluation.

Phase 6 · Iterate

Intent

Hand the program off as a self-iterating system, not as a static deliverable. By Phase-6 gate (W26) the in-house team owns: (a) a decision-rule playbook that fires sprints automatically from L1–L4 signals; (b) a 4-week content-refresh sprint template slotted between cohort starts; (c) a hand-off pack covering SME continuity, dashboard ownership, v2 ES + DE scope; (d) a CRO-ready 6-month review deck for K.M.
