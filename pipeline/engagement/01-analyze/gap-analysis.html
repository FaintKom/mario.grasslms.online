Why this artefact: Crosses Dirksen's 5-dimension behaviour-gap model (Knowledge / Skill / Motivation / Environment / Communication, Dirksen 2015) against each persona, then applies Cathy Moore's Action-Mapping decision rule (Moore 2017) — for each gap, is the fix training, environment, or both? Gap analysis · Dirksen × Cathy Moore

1 · Framing

Dirksen (2015) frames performance gaps across five dimensions:

  • Knowledge — does the rep know the thing?
  • Skill — can the rep do the thing, fluently, under load?
  • Motivation — does the rep want to do the thing?
  • Environment — does the rep's context (tools, comp, manager, team) allow / reward / enable the thing?
  • Communication — does the rep get feedback on whether they're doing the thing?

Moore (2017)'s overlay: for each gap, ask whether training is the right intervention. If the rep would do it tomorrow given will + reward, it isn't training. Otherwise, training is a candidate — alongside or instead of an environment fix.

4 · Cathy Moore "is it training?" pass — explicit decisions

For every gap above, the question "would the rep do this tomorrow if they wanted to and were rewarded for it?" gives:

Gap If yes (= not training) If no (= training)
M1 diagnostic opener Would not — they don't know how to pick the diagnostic (knowledge + skill) Training
M2 acknowledge Would not — argumentation is the reflex (skill) Training
M3 calendar-close Partly would — calendar-in-second-tab is a tool habit; rest is comp-distorted Training + environment
Manager weekly cadence Some would, some wouldn't — varies by pod Environment (rubric, cadence floor) — calibration not training course
Comp SPIF chase on stale leads Yes — reps would chase the SPIF tomorrow because they're rewarded for it Environment only — RevOps comp redesign
Marketing lead quality decline Yes if marketing fixes it; no reward-shift for the rep Environment only — CMO conversation
Leaderboard motivation damage Removed by simply turning off the leaderboard Environment only — comms decision

7 · The honest summary

The brief asks whether ramp compression and retention can be solved by training. The Dirksen × Moore pass says:

  • Training fully solves about 60 % of the gap — the knowledge + skill gaps in M1/M2/M3 + ICP + props + regulatory deflection.
  • Training-adjacent (the rubric) addresses another ~25 % — the manager-cadence variance which the brief's own §9 shows directly correlates with ramp.
  • The remaining ~15 % is environment-only — lead quality + comp SPIF + manager hiring — and K.M. has pre-acknowledged this.

These percentages are qualitative-derived from the brief evidence, not measured. They are sponsor-validation candidates in W2 with K.M. Phase 5 reports actuals from L3/L4 telemetry.
