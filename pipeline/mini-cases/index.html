Three tight single-keystone SCORM 1.2 builds — compliance / phishing, call-centre soft-skills, ERP rollout. Each one ships as a working package ready for any SCORM 1.2 LMS. Built to show the same instructional-design discipline scales down to a focused deliverable.

The FinTechCard flagship case shows depth — one engagement, six ADDIE phases, six SCORM modules, 16 frameworks. These three cases show range — how the same discipline scales down to a single keystone moment, picked because it's where the business actually loses money or time today.
